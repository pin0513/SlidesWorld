# 案例研究：The Sword in the Stone

> 展示 slide-maker + english-teaching-expert 完整整合的範例

---

## 專案概述

| 項目 | 規格 |
|------|------|
| 主題 | 石中劍（亞瑟王傳說）|
| 類型 | 英文互動繪本投影片 |
| 目標年齡 | 5 歲 (Level 1) |
| 頁數 | 16 頁（14 故事 + 2 複習）|
| 核心單字 | 12 個 |
| 句型 | 4 個 |
| 迭代 | 8 個版本（v1→v8）|

## 完整專案位置

```
~/slides/sword-in-stone/
```

## 使用的技能

### english-teaching-expert

| Agent | 貢獻 |
|-------|------|
| Content Planner | 12 個核心單字、4 個句型規劃 |
| Education Expert | Level 1 難度驗證 |
| Fun Curator | 5 歲趣味性審查 |
| Story Writer | 石中劍故事改編 |
| Art Director | 童話風格定義 |
| Storyboard Artist | 14 頁分鏡設計 |
| Animation Designer | 分層動畫系統 |
| Audio Producer | TTS 配音統籌 |
| Pronunciation Coach | 發音準確性審查 |
| Content QA | 最終品質驗收 |

### slide-maker 系統

| 技能 | 貢獻 |
|------|------|
| slide-image | Gemini AI 生成 18 張角色圖 + 4 張背景圖 |
| slide-maker | HTML/CSS/JS 互動繪本製作 |
| slide-reviewer | 6 維度品質審核 |

## 版本演進

| 版本 | 焦點 | 學到的教訓 |
|------|------|-----------|
| v1 | 基礎翻頁互動 | MVP 先行 |
| v2 | 全螢幕動畫 | 沉浸式體驗很重要 |
| v3 | 分層 + 對話框 | 角色對白增加互動感 |
| v4 | 對話框優化 | 行動裝置閱讀體驗 |
| v5 | 真正分層（去背） | 角色可重用，減少資源 |
| v6 | 可讀性優化 | 字體、語速、圖片需預規劃 |
| v7 | 佈局修正 + 複習頁 | 100% 圖片使用率 |
| v8 | CSS 層級標準化 | z-index 系統化管理 |

## 核心學習內容

### 12 個單字

sword, stone, king, knight, big, strong, pull, small, boy, kind, crown, try

### 4 個句型

- I can ___
- I can't ___
- Let me try!
- You are the ___!

## 技術亮點

- **分層架構**：9 層 z-index 系統
- **AI 去背**：Gemini 生圖 + rembg 去背
- **互動詞彙卡**：點擊學習 KK 音標
- **TTS 配音**：macOS 內建 + 可升級 ElevenLabs
