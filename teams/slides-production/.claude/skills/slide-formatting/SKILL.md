# Slide Formatting Skill

## Capability

Apply consistent visual formatting to slide elements including typography, colors, spacing, alignment, and layout. Ensure all slides meet visual quality standards and match design specifications. Useful for Slide Builder and Visual Designer.

## When to Use

- Slide Builder needs to apply design specifications consistently
- Visual Designer is documenting formatting standards
- Quality Reviewer needs to verify formatting consistency
- User requests visual refinement or consistency fixes

## Input Requirements

**Required**:
- Design specifications (color palette, typography, spacing)
- Slides to format (editable Google Slides or similar)

**Optional**:
- Brand guidelines (MAYO, custom brand)
- Specific formatting issues to address
- Layout preferences (minimal, data-heavy, visual-heavy)

## Process

### Step 1: Review Design Specifications
- Understand color palette, typography, spacing standards
- Note any brand guidelines (MAYO, custom)
- Identify layout preferences

### Step 2: Apply Typography
- Set slide title font, size, color, alignment
- Set body text font, size, color, line height
- Apply consistent bullet styles
- Format captions and sources

### Step 3: Apply Colors
- Use primary color for headings and key elements
- Use secondary color for body text
- Use accent color for highlights
- Ensure background color consistency

### Step 4: Apply Spacing and Alignment
- Set consistent margins (top, bottom, left, right)
- Apply element spacing (title to content, between bullets)
- Align elements to grid
- Ensure visual balance

### Step 5: Apply Branding Elements
- Insert logo in consistent position
- Add slide numbers in consistent position
- Apply footer or header if specified

### Step 6: Verify Consistency
- Check all slides for consistent formatting
- Fix any deviations or inconsistencies
- Validate against visual quality standards

## Quality Standards

### Consistency
- All slide titles same font, size, color
- All body text same font, size, color
- Spacing consistent across slides
- Logo position consistent across slides

### Alignment
- Elements aligned to grid (not random placement)
- Text left-aligned (unless specified otherwise)
- Images aligned with layout standards

### Visual Quality
- Font sizes meet minimums (18pt body, 28pt titles)
- Color contrast meets WCAG AA (4.5:1 ratio)
- No orphaned words or widows

## Common Use Cases

1. **Initial Formatting**: Applying design specs to newly built slides
2. **Consistency Fixes**: Correcting formatting inconsistencies across slides
3. **Brand Compliance**: Applying brand guidelines (MAYO, custom)
4. **Visual Refinement**: Improving spacing, alignment, and polish
5. **Template Application**: Applying template formatting to content slides

## Limitations

- Requires editable slide access (cannot format locked or PDF slides)
- Some fonts may not be available (use fallbacks)
- Complex layouts may require manual adjustments
