# 數據報表投影片 Web 示範

## 📊 專案說明

這是一個完整的數據報表投影片 Web 示範，展示：
- 50個學生的多科目成績分析
- 雲端架構系統流程圖 (A → B1 → C → D → A)
- 財產分佈圖
- 系統架構圖（前後端、雲端/地端）

## ✨ 特色功能

✅ **單一 HTML 檔案** - 13KB，雙擊即可開啟，無需伺服器
✅ **互動式圖表** - Chart.js 動態圖表，可 hover 查看數據
✅ **SVG 向量圖** - 流程圖與架構圖，手繪 SVG，保證顯示
✅ **鍵盤導覽** - 支援 ← → Space 鍵切換
✅ **響應式設計** - PC + Mobile 雙平台支援
✅ **自動縮放** - 不同螢幕尺寸自動適配
✅ **零外部依賴** - 完全自包含（除 Chart.js CDN）

## 🚀 使用方式

### 方式 1: 直接開啟（推薦）
```bash
# 雙擊 index.html 即可在瀏覽器中開啟
open output/index.html
```

### 方式 2: 使用簡易伺服器
```bash
cd output
python3 -m http.server 8000
# 瀏覽器開啟 http://localhost:8000
```

## 📊 投影片內容

1. **封面** - 數據報表示範 + 統計卡片
2. **成績分析 1** - 各科目平均分數（條形圖）
3. **成績分析 2** - 五科目能力分析（雷達圖）
4. **流程圖** - 雲端架構循環流程（SVG 手繪）
5. **財產分佈** - 資產配置圓餅圖
6. **系統架構** - 前後端、雲端/地端架構（SVG 手繪）
7. **總結** - 完成頁面

## 🎯 圖表說明

### Chart.js 互動圖表
- **條形圖**：50個學生、5個科目的平均分數
- **雷達圖**：科目能力分析（國文/英文/數學/社會/自然）
- **圓餅圖**：財產分佈（房地產/股票/現金/債券/其他）

### SVG 手繪圖表
- **流程圖**：A → B1 → C → D → A（循環流程）
  - 藍色節點 A：起始節點
  - 綠色節點 B1：處理節點
  - 橘色節點 C：資料節點
  - 紫色節點 D：輸出節點

- **系統架構圖**：分層架構
  - 前端層：React UI + Vue.js Dashboard
  - 後端層：Node.js API + Express Gateway
  - 雲端服務：AWS S3/Lambda/RDS
  - 地端服務：Local Cache + Edge Server
