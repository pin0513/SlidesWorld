## 結構模板

### 15 分鐘標準版 (10-12 頁)

```
1. Hook：震撼開場 (1 min)
   - 一個問題
   - 一個畫面
   - 一句話讓人停下來

2. 我是誰 (2 min)
   - 簡短自我介紹
   - 為什麼我來講這個
   - 建立可信度

3. 故事開始：曾經的我 (2 min)
   - 起點在哪裡
   - 當時的狀態
   - 讓聽眾認同「我也是這樣」

4. 挑戰來臨：低谷時刻 (3 min)
   - 遇到什麼困難
   - 情感的真實描述
   - 最黑暗的時刻

5. 轉捩點：領悟的瞬間 (3 min)
   - 什麼讓我改變
   - 核心洞見（1-3 個）
   - 「啊哈！」時刻

6. 應用：如何實踐 (2 min)
   - 具體可做的事
   - 聽眾可以嘗試的
   - 小步驟開始

7. 金句收尾 (2 min)
   - 回扣開頭
   - 一句話總結
   - 情感高潮
```

### 25 分鐘深度版 (18-22 頁)

擴展標準版：
- 更詳細的背景故事
- 多個低谷時刻或挑戰
- 2-3 個核心洞見分開講解
- 每個洞見配一個故事/例子
- 互動環節

## 五種經典敘事框架

### 1. 問題-領悟-行動 (PIA)

```
Problem: 我曾經...（問題/困境）
Insight: 直到我發現...（領悟/轉變）
Action: 現在你可以...（行動呼籲）
```

**範例**：
> P: 我曾經每天工作 16 小時，以為這才叫努力
> I: 直到我發現，真正的生產力不是時間，是專注
> A: 試試「90 分鐘專注塊」，你的人生會不一樣

### 2. Before-After-Bridge (BAB)

```
Before: 改變之前的狀態
After: 改變之後的狀態
Bridge: 連接兩者的關鍵
```

**範例**：
> Before: 五年前，我是個害怕演講的工程師
> After: 現在，我站在這裡和你們分享
> Bridge: 一個簡單的練習改變了一切

### 3. 黃金圈 (Why-How-What)

```
Why: 為什麼這很重要
How: 如何做到
What: 具體是什麼
```

**範例**：
> Why: 每個人都值得被聽見
> How: 透過刻意練習表達
> What: 從每天寫 200 字開始

### 4. Star Moment 設計

每個演講都需要一個「Star Moment」——聽眾回家會記住的那一刻

**類型**：
- 震撼數據
- 動人故事
- 視覺衝擊
- 互動體驗
- 金句名言

### 5. 回扣技巧 (Callback)

開頭拋出的問題/畫面，在結尾用答案/轉變收尾

```
開頭：「你有沒有在凌晨三點，突然覺得人生沒有意義？」
...（整個演講）...
結尾：「現在，當凌晨三點醒來，我知道那是身體在提醒我休息。
       不是人生沒有意義，是我還沒找到。」
```

## 敘事技巧

### 開場方式

**A. 問題開場**
> 「你有沒有想過，為什麼我們總是在追逐，卻很少停下來問：追到了，然後呢？」

**B. 故事開場**
> 「三年前的這一天，我站在辦公室的窗邊，第一次認真想過：如果跳下去...」

**C. 數據開場**
> 「根據統計，有 70% 的人對自己的工作感到不滿。今天在座的各位，你是那 70% 嗎？」

**D. 衝突開場**
> 「我爸告訴我：穩定最重要。我媽告訴我：聽你爸的。但我今天要告訴你們另一個故事。」

### 情感曲線控制

**低點的力量**
- 不要害怕展示脆弱
- 真實的低谷更能引起共鳴
- 具體的細節（時間、地點、感受）

**轉折的設計**
- 不是「突然我就好了」
- 而是「有一天，我看到/聽到/讀到...」
- 轉折要有觸發點

**高潮的收尾**
- 回扣開頭
- 一句可被記住的話
- 留下餘韻

### 金句設計原則

1. **簡短有力**：10 字以內最佳
2. **對比張力**：「不是 A，而是 B」
3. **可被引述**：聽眾能記住、能轉述
4. **個人色彩**：融入你的經歷

**範例**：
- 「你不需要很厲害才開始，但你需要開始才會變厲害。」
- 「失敗不是終點，放棄才是。」
- 「人生沒有白走的路，每一步都算數。」
- 「最難的不是做選擇，是接受選擇的後果。」

## 經典 TED Talk 結構參考

### Simon Sinek - Start with Why

```
1. 觀察：大多數公司說 What、How，很少說 Why
2. 模式：成功的領袖都從 Why 開始
3. 案例：Apple、Martin Luther King Jr.
4. 原理：黃金圈理論
5. 應用：你也可以從 Why 開始
```

### Brené Brown - The Power of Vulnerability

```
1. Hook：研究員的故事
2. 發現：脆弱是連結的核心
3. 掙扎：我不想接受這個發現
4. 崩潰與重建：個人經歷
5. 洞見：脆弱不是軟弱，是勇氣
6. 呼籲：讓自己被看見
```

### Amy Cuddy - Your Body Language Shapes Who You Are

```
1. 開場：身體語言影響他人看我們
2. 研究：但它也影響我們看自己
3. 實驗：高權力姿勢的效果
4. 個人故事：車禍後的掙扎
5. 應用：Fake it till you become it
6. 結尾：不只是 fake it，是 become it
```

## 常見陷阱

| 陷阱 | 問題 | 解法 |
|------|------|------|
| 說教感太重 | 聽眾覺得被教訓 | 多說「我」的故事，少說「你應該」 |
| 故事太長 | 失去重點 | 5-3-1 法則：5 分鐘故事只保留 3 個細節，1 個核心訊息 |
| 太正面 | 不可信 | 承認低谷和脆弱，真實更動人 |
| 沒有應用 | 聽完不知道能做什麼 | 給 1-3 個具體可執行的小步驟 |
| 結尾太弱 | 虎頭蛇尾 | 準備一句金句，回扣開頭 |
